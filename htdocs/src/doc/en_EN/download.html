Getting with cvs

Anonymous CVS Access

This project's CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. The module you wish to check out must be specified as the unique. When prompted for a password for anonymous, simply press the Enter key.

cvs -d :pserver:anonymous@cvs.luaforge.net:/cvsroot/unique login
cvs -d :pserver:anonymous@cvs.luaforge.net:/cvsroot/unique checkout unique
