Building on linux platform with gcc compiler

Requirements

For building lUniquE you need GNU Make (>=3.81) and GNU C++ compiler (>=3.4).

Some modules require external libraries :

  • lua5.1 - a virtual machine
  • toluaxx5.1 - a tool to integrate C++ and lua (see project toluaxx on luaforge.net)
  • OpenGL® - 3D graphics API
  • OpenAL® - 3D sound API
  • libjpeg - JPEG 2D image library
  • libpng - PNG 2D image library
  • freetype - FreeType TTF font library

    • Configure

    Check if pkg-config tool is installed in your system. In your linux distribution this tool must installed by default.

    Extract project sources and edit options in a config file ( in the project sources root directory ) to correspond your system parameters.

    For instance,

    ## OS Platform
    ## linux|win32
    PLATFORM=linux

    ## Compiler
    ## gcc|mingw
    COMPILER=gcc

    ## Design
    ## modules|&monolith|&runable
    DESIGN=modules monolith runable

    ## Mode
    ## test|real
    MODE=real

    ## Debug
    ## enable|disable
    DEBUG=disable

    ## Main name
    ## lunique
    NAME=lunique

    ## Optimize
    ## disable|enspeed|ensize
    OPTIM=enspeed

    Make calls

    If all configured correctly, you can use the following command for building:

    make all
