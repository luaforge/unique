lUniquE - is a lua based virtual reality engine.

Our engine has a smart modular structure. Each module provides his functionality through lua interface. This system may be used to create interactive applications (based on lua).

Currently available modules:

  • unbase - a base functionality module (time, CPU, system calls routines);
  • unmath - library for three-dimensional maths (contains the following types: vec2, vec3, vec4, mat3, mat4, quat, also some more useful operations with this types and other functions);
  • undata - a data manager module (based on repository conception and resources implementation);
  • unogl - an OpenGL® graphics library integration module (currently available : OpenGL 2.1, OpenGL extensions, keyboard and mouse input);
  • unvis - library for TEXTURE, MATERIAL, SHADER handling;
  • unoal - an OpenAL® audio library integration module;

Coming soon:

  • unobj - virtual reality basic objects, such as OBJECT (a base object for any other scene objects), GROUP (semantic group of any objects), CAMERA (scene camera implementation), LIGHT (light source object) and SCENE;
  • ungeom - basic geometry objects, such as MESH (triangle models), HMAP (height map), and others;
  • uneff - particle systems, fakes and etc.
  • unsnd - sound objects, such as LISTEN (scene listener) and SOUND (scene sound source);
  • ungui - GUI elements (FONT, WIDGET, CURSOR, TEXT, WINDOW).
