Общие требования

Некоторые модули требуют следующие библиотеки:

  • lua5.1 - виртуальная машина
  • toluaxx5.1 - средство для интегрирования C++ и lua (см. проект toluaxx on luaforge.net)
  • OpenGL® - 3D графический API
  • OpenAL® - 3D аудио API
  • libjpeg - библиотека работы с JPEG
  • libpng - библиотека работы с PNG
  • freetype - библиотека работы с шрифтами TTF

    • Сборка на платформе linux с использованием компилятора gcc

    Требования

    Для сборки lUniquE вам потребуется GNU Make (>=3.81) утилита сборки и GNU C++ (>=3.4) компилятор.

    Удостоверьтесь что pkg-config установлен на вашей системе. В большинстве дистрибутивов Linux этот инструмент может быть установленным по умолчанию.

    Конфигурирование

    Распакуйте исходные тексты программы и отредактируйте опции в файле config ( в корневой директории исходного кода проекта ) для обеспечения конфигурации сборки, требуемой для вашей системы.

    Для примера,

    ## Операционная система/ Платформа
    ## linux|win32
    PLATFORM=linux

    ## Компилятор
    ## gcc|mingw
    COMPILER=gcc

    ## Дизайн
    ## modules(модульно)|&monolith(монолитно)|&runable(запускаемо)
    DESIGN=modules monolith runable
    ## можно комбинировать как угодно

    ## Режим
    ## test|real
    MODE=real
    ## для сборки всегда должен стоять real.
    ## если вы хотите протестировать работу сборки
    ## установите test, вы увидете вывод команд,
    ## которые будут вызваны в real режиме

    ## Опции отладки
    ## enable|disable
    DEBUG=disable

    ## Главное имя
    ## lunique
    NAME=lunique
    ## будет использовано для поименования модуля
    ## для дизайна monolith и runable

    ## Опции оптимизации
    ## disable|enspeed|ensize
    OPTIM=enspeed
    ## по умолчанию оптимизация на скорость

    Вызов make

    Если все опции конфигурации установлены корректно, вы можете использовать для сборки следующую команду:

    $ make all
