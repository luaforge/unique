Этот движок имеет развитую модульную структуру. Каждый модкль обеспечивает свою функциональность посредством интерфейса с Lua. Это програмное обеспечение может быть использовано для создания интерактивных 3D приложений (базирующихся на Lua).

В настоящее время доступны следующие модули:

  • unbase - базовый модуль (время, CPU, системные вызовы);
  • unmath - библиотека трехмерной математики (содержит следующие типы данных: vec2, vec3, vec4, mat3, mat4, quat, а также множество полезных операций с этими типами и другие функции);
  • undata - менеджер данных (базирован на концепции репозиториев и работе с ресурсами);
  • unogl - модуль работы с библиотекой 3D графики OpenGL® (в настоящий момент обеспечивает поддержку: OpenGL 2.1, OpenGL расширений) Модуль также предоставляет инструментарий работы устройствами ввода оконной системы (в настоящий момент доступны клавиатура и мышь);
  • unvis - библиотека визуальных средств (предоставляет объекты: TEXTURE, MATERIAL, SHADER;
  • unoal - модуль работы с библиотекой 3D звука OpenAL® ;

Скоро будут:

  • unobj - базовые объекты виртуальной реальности, такие как OBJECT (базовый объект для других объектов сцены), GROUP (логические группы из других объектов), CAMERA (объект камера сцены), LIGHT (объект источника света) и SCENE (собственно сцена);
  • ungeom - базовые геометрические объекты, такие как MESH (сеточные модели), HMAP (карты высот), базовые примитивы и другие;
  • uneff - системы частиц, вспышки и пр.
  • unsnd - звуковые объекты, такие как LISTEN (слушатель сцены) и SOUND (источник звука сцены);
  • ungui - GUI элементы визуального интерфейса (FONT, WIDGET, CURSOR, TEXT, WINDOW).
