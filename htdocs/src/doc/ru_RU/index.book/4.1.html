lUnique имеет полностью аппаратный рендерер, работа которого базируется на OpenGL API.

Рендерер активно использует аппаратные возможности видео подсистемы, доступ к которым предоставляется API OpenGL. Рендерер состоит из двух тесно взаимосвязанных частей:

  • Менеджер целей рендеринга - отвечающий за выполнение правил рендеринга в соответствующие целевые буферы.
  • Мэнеджер рендеринга геометрии - осуществляющий непосредственный вывод геометрии и поддерживающий иерархию объектов рендеринга.
