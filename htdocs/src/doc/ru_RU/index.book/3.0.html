Для обеспечения полного соответствия требований приведем несколько рекомендаций по созданию ваших моделей:

  • Центры объектов располагать в их центре
  • Для заведомо одинаковых объектов использовать дублирование без дублирования геометрии (M-d вместо S-d)
  • Выравнивать углы и координаты (например если 89.8* заведомо 90*)
  • Использовать отделение поверхностей методом Split где это необходимо например для определения острых углов между поверхностями (нормали автоматически отделяются при методе их рассчета Smooth)
  • Стараться не объединять в одном объекте два отдельных (две отдельные друг от друга геометрии)
  • Именовать объекты. При этом использовать логическую группировку
  • Следить за тем, куда повернуты нормали

Эти рекомендации являются результатом выводов, сделанных в ходе конференции по моделингу сцен и объектов для lUnique. Это не в коем случае не полный перечень рекомендаций и в дальнейшем он будет расширен и дополнен.
